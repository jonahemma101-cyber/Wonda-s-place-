## 📌 Project Overview

This website is designed for **Wonda’s Place** to give customers a clean, modern way to:

- See food images clearly
- Place orders instantly via WhatsApp
- Locate the restaurant easily on Google Maps

The goal is simple:
> Turn online visitors into paying customers.
